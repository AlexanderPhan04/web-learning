| Ký hiệu   | Viết đầy đủ   | Nghĩa                                                        |
| --------- | ------------- | ------------------------------------------------------------ |
| **S**     | Subject       | **Chủ ngữ** – người hoặc vật **thực hiện hành động**         |
| **V**     | Verb          | **Động từ** – hành động hoặc trạng thái                      |
| **N / O** | Noun / Object | **Tân ngữ** – người hoặc vật **chịu tác động của hành động** |

| Cấu trúc           | Câu tiếng Trung | Phiên âm         | Nghĩa tiếng Việt    |
| ------------------ | --------------- | ---------------- | ------------------- |
| **S + V + O**      | 我吃苹果。      | Wǒ chī píngguǒ.  | Tôi ăn táo.         |
| **S + V**          | 他来了。        | Tā lái le.       | Anh ấy đến rồi.     |
| **S + 是 + N**     | 我是学生。      | Wǒ shì xuéshēng. | Tôi là học sinh.    |
| **S + 不 + V + O** | 我不喝茶。      | Wǒ bù hē chá.    | Tôi không uống trà. |
| **S + 在 + N**     | 他在家。        | Tā zài jiā.      | Anh ấy ở nhà.       |

| Thanh điệu      | Cách gõ số        | Ví dụ không dấu | Kết quả |
| --------------- | ----------------- | --------------- | ------- |
| 1 (thanh ngang) | `1`               | ma1             | mā      |
| 2 (thanh lên)   | `2`               | ma2             | má      |
| 3 (thanh uốn)   | `3`               | ma3             | mǎ      |
| 4 (thanh xuống) | `4`               | ma4             | mà      |
| không thanh     | `0` hoặc bỏ trống | ma              | ma      |
